Bonus Quest Plan

Archive Detective · Thousand Token Wood · six merit badges · local-first, no deploy until verified

Verdict: All six badges are achievable, but not with the current Modal + MiniCPM + default Gradio stack alone. The winning stack is one local pipeline: fine-tuned text model → GGUF → llama.cpp → gr.Server custom UI, plus a Hub dataset for traces and a field report.

Unified architecture (hits 4 technical badges)

Build order (local test after each step)

  1. Field Notes — write the report as we go (documents failures: HF deploy, Modal, etc.)
  2. Off-Brand — server.py + static board UI; wire existing engine APIs; verify_local.py still passes
  3. Llama Champion + Off the Grid — scripts/run_llama.sh, base instruct GGUF, OCR→JSON prompt; no cloud inference
  4. Well-Tuned — export ~40–80 (OCR, clue_pack) pairs → SFT → publish model → convert GGUF → swap into llama path
  5. Sharing is Caring — redact tokens from Cursor transcript → public dataset
  6. Only then — HF Space deploy with --confirm-local

Per-badge checklist

🔌 Off the Grid
Achievable
  • Demo runs with ARCHIVE_DETECTIVE_BACKEND=llama + local server
  • No OpenAI / HF Inference API / Modal in user-facing flow
  • LOC fetch stays a dev script, not the Gradio app
  • Ship a GGUF in repo or document one-time hf download then air-gap
🎯 Well-Tuned
Achievable, needs GPU time
  • Train on existing clue packs (heuristic + MiniCPM outputs)
  • Base: Qwen2.5-1.5B-Instruct or Llama-3.2-3B-Instruct (<32B)
  • Publish: nice-bill/archive-detective-clue-sft
  • Submission README links the fine-tuned repo explicitly
🎨 Off-Brand
Achievable
  • Replace gr.Blocks demo with gradio.Server
  • Custom evidence-board HTML + JS client
  • Must look nothing like default Gradio tabs/dropdown chrome
🦙 Llama Champion
Achievable
  • You already have llama-server on PATH
  • Wire vision.py → llama.cpp HTTP or python binding
  • Text-from-OCR is enough; image optional for badge
📡 Sharing is Caring
Easy
  • Source: Cursor agent transcript JSONL for this project
  • Redact HF_TOKEN, paths, .env before upload
  • Dataset: nice-bill/archive-detective-agent-traces
📓 Field Notes
Easy
  • HF blog post or docs/artifacts/field-notes.html
  • Cover: LOC ingest, clue packs, local vs cloud mistakes, badge stack
